## O que é

O Nexus X é um otimizador de sistema para Windows focado em jogos e tarefas pesadas:

| Ferramenta | O que faz |
|---|---|
| **Memória** | Libera RAM em um clique e limpa arquivos residuais do disco |
| **Otimizar Rede** | Reduz latência com 5 módulos aplicados em sequência (Flush DNS, TCP DCA, Nagle, Throttling, Responsividade) |
| **Otimizar Jogos** | Perfis de FPS prontos (Máximo FPS, Balanceado, Competitivo) |
| **Modo Extremo** | Ajustes profundos do Windows: efeitos visuais, plano de energia, apps em segundo plano, indexação, memória virtual e prioridade de CPU |

## Download e instalação

1. Baixe o instalador na página de [Releases](https://github.com/dxntagency/NEXUS-X/releases/latest).
2. Execute o instalador e siga as instruções.
3. O aplicativo pede privilégios de administrador — necessários para aplicar as otimizações de sistema.

## Requisitos

- Windows 10 ou 11 (64 bits)
- Privilégios de administrador
- Licença ativa Nexus X
